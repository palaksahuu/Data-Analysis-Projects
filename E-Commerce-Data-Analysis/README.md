# Superstore Sales Data Analysis

This project performs exploratory data analysis on the Superstore dataset using Python. The goal of the analysis is to understand sales performance, customer behavior, and trends in orders over time.

The dataset is cleaned and transformed using pandas, and time-based features are extracted from order dates to analyze monthly sales trends.

---

## Dataset Overview

The dataset contains retail order information from a superstore. It includes details about customers, products, orders, shipping, and financial metrics.

Total records: 9994 rows  
Total columns: 21 columns

Important columns include:

- Order ID
- Order Date
- Ship Date
- Customer Name
- Segment
- City
- State
- Region
- Category
- Sub-Category
- Product Name
- Sales
- Quantity
- Discount
- Profit
