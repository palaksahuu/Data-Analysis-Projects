# Trader Behavior vs Market Sentiment Analysis

This project explores how trader performance is influenced by market sentiment using Hyperliquid trading data and the Bitcoin Fear & Greed Index.

## Analysis Performed

• Profit vs sentiment
• Leverage usage
• Trade direction behavior
• Win rate analysis
• Top trader identification

## Tools

Python
Pandas
Matplotlib
Seaborn

## Results

The study reveals clear behavioral patterns in trader activity during Fear and Greed market conditions.
